# TypeScript 실무 팁 모음

TypeScript를 사용하면서 알게 된 유용한 팁들을 공유합니다.

## Type Guards

타입 가드를 사용하면 타입을 안전하게 좁힐 수 있습니다.

```typescript
function isString(value: unknown): value is string {
  return typeof value === 'string';
}
```

## Utility Types

TypeScript의 내장 유틸리티 타입들을 활용하면 더 간결한 코드를 작성할 수 있습니다.
